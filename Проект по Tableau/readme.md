Проект выполнен для крупной телекоммуникационной компании для анализа уровня потребительской лояльности (NPS) среди клиентов в России. На основе результатов опроса был разработан интерактивный дашборд в Tableau, который визуализирует итоговый NPS и позволяет исследовать, как лояльность клиентов распределяется по различным пользовательским сегментам (возраст, пол, стаж, тип устройства, география и т.д.).
